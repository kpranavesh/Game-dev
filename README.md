# Us: The Game

A personalized birthday gift game for my wife, built with Python + Pygame.

7 chapters telling the story of how we met, fell in love, got married, and got a dog named Beans.

## How to Play

Double-click `Play Game.command` or run:

```
python3 src/main.py
```

## Chapters

1. **Swipe Right** - Find Krishna on Hinge
2. **Will You Be Mine?** - Set up the perfect proposal scene
3. **The Charles River** - A message in a bottle on kayaks
4. **Good Girl, Beans** - Teach Beans commands (SIT, DOWN, MIDDLE, FREE)
5. **Yarn Crisis** - Catch yarn balls before Beans steals them
6. **Beans at Harris Park** - Endless runner with Beans
7. **Finale**

## Controls

- Arrow keys / WASD to move
- Space / Enter to interact
- Mouse clicks where applicable
- ESC to quit

## Stack

- Python 3.9+
- Pygame 2.5+
- NumPy (for sound generation)
- All art is procedurally drawn, no external assets except two audio files
